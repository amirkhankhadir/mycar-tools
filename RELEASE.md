# Как обновлять плагины (release & update)

> Памятка для мейнтейнера. Установленная копия плагина и рабочий репозиторий — **два разных места**.
> Правка исходника в репо НЕ меняет установленную версию у пользователя, пока не выполнен цикл публикации.

```
~/.claude/plugins/cache/mycar-tools/<plugin>/<version>/   ← замороженный снимок (то, что «работает» у пользователя)
UX-Writing-Guide/plugins/<plugin>/                         ← рабочий исходник (то, что правим)
```

## Цикл разработки (быстрый, без GitHub-круга)
Пока пилим — тестируем **против локальных файлов репо**, переустановка не нужна.
Живой интерактивный тест локальной версии:
```
claude --plugin-dir /Users/amirkhankhadir/Documents/GitHub/UX-Writing-Guide/plugins/mycar-copy
```
→ грузит локальный плагин в сессию как есть, читает живые файлы. Правки видны сразу.

## Цикл публикации (когда правки готовы к раздаче)
1. Правим в `UX-Writing-Guide/plugins/<plugin>/`
2. **Бампаем `version`** в `plugin.json` (напр. `0.1.0 → 0.1.1`) — БЕЗ этого `update` не подтянет (дедуп по версии)
3. `git commit` + `git push`
4. Для Figma-витрины (только `mycar-copy`): пересобрать `figma/mycar-copy.md` и перепубликовать (правило синка правила↔витрина)

## Как обновляется потребитель (дизайнер / другая машина)
```
claude plugin marketplace update mycar-tools   # обновить манифест маркетплейса
claude plugin update mycar-copy                # подтянуть новую версию
```
→ рестарт сессии, чтобы применилось.

Первичная установка с нуля:
```
claude plugin marketplace add amirkhankhadir/mycar-tools
claude plugin install mycar-copy@mycar-tools
```

## ⏰ Когда публиковать `mycar-copy` (release-триггер)
Не гоняем GitHub-круг на каждую правку — публикуем на вехах. Обновить (bump + push + designer update), когда закрыты контент-хвосты:
- [ ] **Глоссарий** подключён (внешний источник, `GLS-*` заработали не только по эвристике)
- [ ] **Бренд-голос / настройки** заполнены: `BRAND_QUIRKS`, политика «!», «ё», раздел «Наши продукты» / бренд-разнобой (mÿoffer / mycar / MyBridge / MyVerify)
- [ ] **Модуль «Числа, валюта, единицы»** доведён (сейчас `NUM-*` — временный минимум)
- [ ] **Открытые вопросы** из `open-questions.md` закрыты (в т.ч. STL-08)
- [ ] Переэвал по `eval-corpus.md` зелёный после всех правок

> Полная детализация хвостов — в `open-questions.md`. Прогресс раздачи — в `how-to-use.md`.
