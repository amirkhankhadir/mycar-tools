# Хендофф разработчикам — как разместить и раздать скилл

Короткая инструкция для того, кто будет размещать это в компании. Разговор с разработчиками сводится
к одной просьбе: **разместить этот пакет в корпоративном репозитории и выдать доступ команде.**

## Что это

Пакет с правилами интерфейсных текстов + **Claude Code skill**, который проверяет, генерирует и
улучшает UI-тексты по нашему гайду. Работает на **уже оплаченных местах Claude Max** — отдельный
API-ключ, бэкенд или бюджет **не нужны**.

Технически это набор Markdown-файлов по спецификации Agent Skills. **Ничего не собирается и не
выполняется**, нет зависимостей, секретов, ключей и внешних вызовов. Разместить = обычный git-репозиторий.

## Что нужно от вас (минимум)

1. Создать **приватный репозиторий** в GitHub-организации компании *(или GitLab/Bitbucket — не принципиально)*.
2. Залить туда содержимое этого пакета. Локально уже готов git-репозиторий с первым коммитом — нужно
   только добавить remote и запушить (команды ниже).
3. Выдать **доступ на чтение** нужной команде: дизайнеры и копирайтеры (пользователи) + те, кто будет
   вести правила.

## Как сотрудник подключает (Claude Code)

```
git clone <адрес-репозитория>
```
Открыть папку репозитория в Claude Code — скилл `/ux-copy` доступен автоматически (лежит в
`.claude/skills/`, это «проектный скилл»).

Обновление правил: `git pull`.

## Кто и как ведёт правила

Правила — в `content-guide.md`. Владелец правит файл → `commit` + `push`. Никакой пересборки и
переустановки: у пользователей всё подтянется через `git pull`.

## Команды для первого пуша (пример для GitHub)

```
git remote add origin git@github.com:<организация>/<репозиторий>.git
git push -u origin main
```

## Про Figma

Вторая витрина (Figma skill) раздаётся **отдельно** — через встроенную публикацию скиллов в Figma на
организацию. Для неё этот репозиторий не нужен; он лишь хранит исходник, из которого собирается
Figma-файл.

## Чего делать НЕ нужно

- Не нужен сервер, хостинг, CI/CD.
- Не нужны API-ключи и секреты.
- Не нужен доступ в интернет во время работы скилла.
